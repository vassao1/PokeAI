selfbot no discord que utiliza uma pipeline de IA p/ reconhecimento de pokémons e automatização de catch do PokéTwo (bot no discord de pokémon). <br>
a fim de curiosidade, a pipeline contém um modelo YOLO que reconhece um objeto pokémon na imagem recebida pelo discord, cria uma bounding box na imagem e faz o crop dela, e depois um modelo EfficientNet faz a classificação de qual pokémon é. <br>
é basicamente isso, quem for usar, use por sua conta e risco, selfbot no discord é banível e autocatch no pokétwo também. <br>
se quiser que seja mais rápido baixa o pytorch com o CUDA driver (se tua placa tiver compatibilidade).
